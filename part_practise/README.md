# Publish
[构建和发布为 Android 应用](https://flutter.cn/docs/deployment/android)
```bash
flutter build apk
```
该命令会打包AndroidAPk文件，会自带`--release`参数
```bash
flutter build apk --split-per-abi
```
该命令会生成单独的abi的包
